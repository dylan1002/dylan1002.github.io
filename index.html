标签（空格分隔）： 未分类

表格操作

获取：
tBodies、tHead、tFoot、rows、cells(单元格)

搜索：
1.字符串比较
2.忽略大小写
3.模糊搜索 search
4.多关键字 split

<script>

window.onload=function()
{
oTab=document.getElementById(‘tab1’);
oBtn=document.getElementById(‘btn1’);
oTet=document.getElementById(‘name’);

oBtn.onclick=function()
{ 
   for(var i=0;i<oTab.tBodies[0].rows.length;i++)
   {
     var sTab=oTab.tBodies[0].rows[i].cells[1].innerHTML.toLowerCase();  //大小忽略
     var sTet=oTet.value.toLowerCase();
     var arr=sTet.split(' ');   //多个关键字 用空格隔开

     oTab.tBodies[0].rows[i].style.background='';  
     for(j=0;j<arr.length;j++)
     {
     if(sTab.search(arr[j])!=-1)    //模糊搜索
     {
       oTab.tBodies[0].rows[i].style.background='yellow';
     }    
     }
   }  
}

}

4.artcle和section的区别。
article元素可以看成是一种特殊类型的section元素，它比section元素更强调独立性。即section元素强调分段或分块，而article强调独立性。具体来说，如果一块内容相对来说比较独立的、完整的时候，应该使用article元素，但是如果你想将一块内容分成几段的时候，应该使用section元素。另外，在HTML5中，div元素变成了一种容器，当使用CSS样式的时候，可以对这个容器进行一个总体的CSS样式的套用。
