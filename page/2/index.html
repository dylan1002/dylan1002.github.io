DOM高级操作

标签（空格分隔）： 未分类

表格操作

获取：
tBodies、tHead、tFoot、rows、cells(单元格)

搜索：
1.字符串比较
2.忽略大小写
3.模糊搜索 search
4.多关键字 split

<script>

window.onload=function()
{
oTab=document.getElementById(‘tab1’);
oBtn=document.getElementById(‘btn1’);
oTet=document.getElementById(‘name’);

oBtn.onclick=function()
{ 
   for(var i=0;i<oTab.tBodies[0].rows.length;i++)
   {
     var sTab=oTab.tBodies[0].rows[i].cells[1].innerHTML.toLowerCase();  //大小忽略
     var sTet=oTet.value.toLowerCase();
     var arr=sTet.split(' ');   //多个关键字 用空格隔开

     oTab.tBodies[0].rows[i].style.background='';  
     for(j=0;j<arr.length;j++)
     {
     if(sTab.search(arr[j])!=-1)    //模糊搜索
     {
       oTab.tBodies[0].rows[i].style.background='yellow';
     }    
     }
   }  
}

}

3.25 Javascript基本知识（语法）

标签（空格分隔）： JavaScript

Js的语法与C语言等有很多相似之处，我只记录有差异的，或者比较容易错的。

变量：

javascript的变量类型是松散类型的（var）,可以用来保存任意类型的数据。

数据类型：

javascript的数据类型有
ECMAScript中有5种简单数据类型（也称为基本数据类型）：Undefined、Null、Boolean、Number和String。还有1种复杂数据类型——Object，其本质是一组 无序 的名值对组成
Number类型：
ECMAScript能保存的最小数值保存在Number.MIN_VALUE中，最大数值保存在Number.MAX_VALUE中
+（-）Infinity表示无穷保存在Number.NEGATIVE_INFINOTY和Number.POSITIVE_INFINITY中。
isNaN(a)函数可以确定一个参数a是不是数值，是数值返回false。

String类型：
把一个值转换为字符串的方法1.toString（）；2.

操作符：

+：如果两个操作数都是字符，则将两个字符串连接起来
如果只有一个操作数是字符串，则将另外一个操作数转换成字符串再将他们拼接

全等（===）：它指在两个操作数在未经转换的情况下就相等的情况 Null===Undefined返回False

语句：

for…in 语句用于遍历数组或者对象的属性（对数组或者对象的属性进行循环操作）。
label语句可以在代码中添加标签，一般配合break、continue使用。。

with语句将代码的作用域设置到一个特定的对象中。
