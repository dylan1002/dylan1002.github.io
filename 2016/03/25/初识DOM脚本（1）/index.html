1.nodeValue 属性设置或返回指定节点的节点值。

例子：
function myFunction()
{
var c=document.getElementsByTagName(“BUTTON”)[0];
var x=document.getElementById(“demo”);
x.innerHTML=c.firstChild.nodeValue;
}

2.动态创建HTML

1.document.write()：
write() 方法可向文档写入 HTML 表达式或 JavaScript 代码。
可列出多个参数(exp1,exp2,exp3,…) ，它们将按顺序被追加到文档中。
为了避免使用〈script〉标签，所以避免使用document.write方法

<div id="testdiv"> </div>
